## Responsibility

This component handles the step-by-step educational cycle:
- `LessonIntro.jsx` presents an introductory tutorial with slides introducing the specific math standard using clear, user-friendly language and solved examples before practice begins.
- `QuestionView.jsx` displays the interactive practice questions, presenting inputs (text fields, multiple choice, coordinate selection) and submitting student answers.
- `ExplanationView.jsx` is triggered when a student submits an incorrect response. It displays a structured, detailed math-focused walkthrough showing how to solve the specific question step-by-step.
