## Responsibility

This module serves as the primary data store and logic core for curriculum-aligned question generation:
- It defines the learning slides and explanatory walkthroughs for each of the five Common Core standards.
- It implements the dynamic question generator functions that dynamically produce randomized math problems customized for three levels of difficulty (`easy`, `medium`, and `hard`).
- `index.js` acts as the single export aggregator for all topics.
